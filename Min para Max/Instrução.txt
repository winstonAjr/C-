Fazer um programa que leia uma string e converta todos os caracteres desta string para maiúsculo. Depois de convertida a string, exibir o resultado na tela.
Dicas: (1) Uma string de caracteres é um vetor do tipo char, terminado pelo caractere ‘\0’; (2) O código ASCII da letra ‘a’ (minúsculo) é 97 e o código da letra ‘A’ (maiúsculo) é 65, o código da letra ‘z’ é 122 e o código da letra ‘Z’ é 90; (3) Cuidado para não acabar convertendo aquilo que não é nem texto e nem está em minúsculas!
